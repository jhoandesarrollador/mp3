# Reproductor de Música en Java

Este es un reproductor de música básico desarrollado en Java. Utiliza las bibliotecas **JLayer** para la reproducción de audio y **JAudioTagger** para leer los metadatos de las canciones. La interfaz gráfica está construida con **Swing**.

## Características

- Reproducción de archivos MP3.
- Interfaz gráfica intuitiva con controles de reproducción (play, pause).
- Visualización del título y artista de la canción.
- Soporte para cargar canciones desde el explorador de archivos.

## Requisitos

- **Java Development Kit (JDK)**: Versión 8 o superior.
- **Dependencias**:
    - JLayer (para reproducción de audio).
    - JAudioTagger (para leer metadatos de archivos de audio).
